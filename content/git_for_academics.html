Concepts and tools

What is version control?

A way to track changes made to files

Benefits of version control

  • collaboration
  • versioning
  • rolling back
  • understanding context
  • backup

Simplify your workflow by having fewer files to keep track of.

Git basics

Git: open source tool installed on your local computer to track changes made to a set of files (referred to as a git repository)

GitHub: a popular website for hosting and sharing Git repositories

Git commands consist of verbs (init, add, commit) preceded by the word git and appended by qualifiers.

The basic workflow starts with a folder and a set of files in a folder.

  • git init - tells git to pay attention to a folder
  • git add . - includes all of the files in the folder
  • git commit -m "a message" - a commitment; a snapshot in time
Git init creates an invisible tracking file. Selective file tracking is not always useful but is a feature too. Git can pay attention to specific files rather than the whole thing.

When working with Github your workflow is a bit different because you need to add and commit but also store changes away from your local computer.

  • git clone - makes a copy of a repository from github on your local computer
  • git pull - pulls "down" the most recent changes before adding new changes
  • git push - after adding and committing, you push changes back up to github

Academic projects have many types of files.

Some are binary and some are non-binary

Non-binary files can be opened with a text editor like Atom, Sublime, or Notepad++. Eg. a text (.txt) file.

Binary files are a bit more complex. Eg. a .pdf, Word .docx, or an image.

Git tracks both binary and non-binary files with one difference:

For non-binary files git remembers the exact information that changed line by line so it doesn't need to duplicate the file.

For binary files git saves a snapshot of the whole file.

Which files are Binary and which are non-Binary?

  • text files (.txt, .csv, .md)
  • PDFs (.pdf)
  • spreadsheets (.xlsx)
  • images (.png, .jpeg, .tiff)
  • Word documents (.doc, .docx)
